# 🚗 Rachou — Carona Compartilhada

O **Rachou** é uma aplicação de carona compartilhada desenvolvida com o objetivo de conectar motoristas e passageiros que desejam compartilhar trajetos de forma prática, econômica e colaborativa. Ao dividir custos de combustível e pedágios, a plataforma promove uma alternativa inteligente e sustentável para a mobilidade urbana e rodoviária.

Este projeto faz parte do Projeto Integrador (PI) desenvolvido durante o bootcamp da **Generation Brasil**.

## 🛠️ Tecnologias e Ferramentas

O front-end do projeto foi estruturado utilizando tecnologias modernas focadas em performance, tipagem estática e padronização de código:

*   **React** (com TypeScript) — Biblioteca para construção de interfaces componentizadas e fortemente tipadas.
*   **Vite** — Build tool ultra-rápida para o ambiente de desenvolvimento.
*   **Tailwind CSS** — Framework utilitário para estilização rápida e responsiva.
*   **Oxlint** — Linter de alta performance focado na qualidade e consistência do código.

## 🎨 Identidade Visual (Paleta de Cores)
